# Programming

        **Level:** AS Level · **Primary assessment:** Paper 2  
        **Key concepts (Scheme of Work):** KC1, KC2

        ## Overview

        Writing structured pseudocode with correct constructs, modular design, and clear interfaces.

        ## Learning objectives (syllabus-mapped)

        - Implement pseudocode from flowchart/structured English.
- Declarations, assignment, expressions, console I/O; built-in/library routines as provided in exam.
- IF/ELSE (nested), CASE; count-controlled, post-condition, pre-condition loops; justify loop choice.
- Procedures and functions; parameters by value/reference; headers, interfaces, arguments, return values.
- When to use procedures vs functions; write efficient pseudocode.

        - **Scheme of Work:** **24** guided hours; integrate **continuously** across the course.

        ## Key terms

        `variable`, `constant`, `procedure`, `function`, `parameter`, `loop`, `selection`

        ## Real-world applications

        - Console utilities
- Data processing
- Embedded logic
